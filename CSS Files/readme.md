## Styles

Files Overview

1. general.css:

    Contains global styles used throughout the entire page:
        •	Base font styles
        •	Body margins & padding
        •	Reusable utility classes
        •	Default layout resets

    This file acts as the foundation for all other CSS files.


2. header.css:

    Styles dedicated to the top navigation bar, including:
        •	YouTube logo section
        •	Search bar & search button
        •	Right-side icons (notifications, profile, etc.)
        •	Header alignment and spacing

    This file handles everything related to the top header UI.


3. sidebar.css:

    Controls the styles of the left sidebar menu, such as:
        •	Menu icons
        •	Hover effects
        •	Sidebar width & layout
        •	Text alignment and spacing

    All vertical navigation elements are managed here.


4. video.css:

    Contains the styles for the video grid area, including:
        •	Video thumbnails
        •	Channel profile pictures
        •	Video titles & metadata layout
        •	Responsive grid design

    This file handles how each video card looks and behaves.
