## youtube.html

Project Files & Folders Explained:
-----------------------------------

1. youtube.html:
-----------------

    This is the main file of the entire project.
    It contains the complete structure of the YouTube homepage clone:
	•	Header (logo, search bar, user icons)
	•	Sidebar navigation menu
	•	Video grid layout
	•	Thumbnail + title + channel info structure
	•	Overall page styling & layout structure

    The code is written to be readable and easy to study.


2.icons/ :
----------

    This folder contains all icons used in the YouTube clone, such as:
        •	Search icon
        •	Microphone icon
        •	Notification bell
        •	Sidebar menu icons
        •	Home / Explore / Subscription icons

    All icons are stored as separate image files to maintain clarity and reusability.

3.image/ :
-----------

    This folder contains:

    ✔️ Thumbnail images
    Used for the video preview cards in the homepage layout.

    ✔️ Channel profile images
    Displayed below each thumbnail next to the video title.

    These images help replicate the visual feel of the actual YouTube homepage.
